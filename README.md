# dylanpagillo.com

Personal portfolio site for Dylan Pagillo — Senior Design Engineer.

Single-file static HTML. No build step. Deployed via Netlify on push to `main`.

## Local preview

Open `index.html` directly in any browser.

## Structure

- `index.html` — the whole site
- `images/` — case study visuals (added as they're ready)
- `resume.pdf` — downloadable resume
